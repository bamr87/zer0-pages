## Overview

zer0-pages is a next-generation content management system that combines:

- **Django Backend**: Robust API with Django REST Framework
- **Jekyll Static Site**: Lightning-fast static site generation
- **AI Integration**: Multi-provider AI support (OpenAI, Anthropic, xAI)
- **GitHub Native**: Deep integration with GitHub for automation
- **Privacy First**: GDPR/CCPA compliant analytics
