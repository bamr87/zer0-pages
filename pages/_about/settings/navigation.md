> [!note] Live-site component
> The interactive "Navigation Editor" widget renders on the published Jekyll site. It provides three tabs — Overview (a navigation tree view), Edit Menus, and Export YAML — for viewing and exporting the site's navigation menu structures.

The Navigation Editor works with these navigation files:

- `main.yml`
- `home.yml`
- `about.yml`
- `docs.yml`
- `posts.yml`
- `quickstart.yml`
- `admin.yml`

Select a navigation file to edit; changes are reflected in the Export YAML tab. Use the tree view in the Overview tab to identify items, then export the full YAML from the Export tab.
