> [!note] Live-site component
> The interactive **Theme Preview** dashboard renders on the published Jekyll site. It combines the theme-controls bar, the component preview gallery, and an appearance override panel (which overrides `--zer0-color-primary` on top of the active skin) so you can preview skins, color modes, and component coverage live. Skin preview is page-level — export permanent changes from the Theme Customizer at `/about/settings/theme/`.

The preview gallery covers the following components:

- Typography
- Buttons
- Alerts
- Cards
- Forms
- Nav tabs
- Badges
- Code blocks
- Links
- Navbar
- Footer
- Table
- List group
- Backgrounds
- Callouts
- Accordion
- Progress & Spinners
- Breadcrumb & Pagination
- Tooltips & Popovers
- Icons
