> [!note] Live-site component
> The interactive Theme Customizer runs on the published Jekyll site. It bundles the theme controls bar, skin previewer, skin editor, palette generator, live CSS-variable editor, color editor, and YAML exporter, none of which render in Obsidian. Open the Theme Preview style guide on the live site to see all components update live with skin and mode changes.

The customizer is organized into the following tools:

- **Skins** — Preview the built-in theme skins.
- **Skin Editor** — Edit any built-in skin or create your own. Adjust gradient colors, preview palettes, tune the SVG filter, then apply live. Click any swatch to copy its hex code, use **Save Custom** to persist your skin in localStorage, and use **Apply Live** to see changes on the page instantly.
- **Palette Generator** — Generate harmonious color palettes from a base color using color theory algorithms (complementary, analogous, triadic, split-complementary, tetradic, and monochromatic harmonies).
- **Live Preview** — Edit Bootstrap CSS variables directly. Changes render instantly on the page and can be exported for permanent use.
- **Color Editor** — Edit theme color values. Changes are reflected in the Export tab.
- **Export** — View, copy, and download the theme configuration as YAML.
