> [!note] Live-site dashboard
> This page is a live statistics dashboard. On the published Jekyll site it renders content analytics and metrics — an overview panel, category and tag breakdowns, and detailed content metrics — from the site's content-statistics data. When that data is unavailable, the site shows a "no data" placeholder instead. The interactive dashboard does not render in Obsidian.
