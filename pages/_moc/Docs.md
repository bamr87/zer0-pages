## Docs by section

```dataview
TABLE rows.file.link AS "Documents", length(rows) AS "Count"
FROM "_docs"
WHERE type = "doc"
GROUP BY choice(file.folder = "_docs", "General", regexreplace(file.folder, "^_docs/", "")) AS Section
SORT Section ASC
```

## Recently updated

```dataview
TABLE choice(file.folder = "_docs", "General", regexreplace(file.folder, "^_docs/", "")) AS Section, description AS Summary
FROM "_docs"
WHERE type = "doc"
SORT default(lastmod, date) DESC
LIMIT 15
```

## By difficulty

```dataview
TABLE rows.file.link AS "Documents"
FROM "_docs"
WHERE type = "doc" AND difficulty
GROUP BY difficulty AS Level
SORT Level ASC
```
