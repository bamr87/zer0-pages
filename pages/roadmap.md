# Roadmap

Development roadmap for the zer0-mistakes Jekyll theme — past releases, current focus, and future plans. All versions follow [Semantic Versioning](https://semver.org/).

> **Single source of truth.** This page and the [README roadmap section](https://github.com/bamr87/zer0-mistakes#-roadmap) are both rendered from [`_data/roadmap.yml`](https://github.com/bamr87/zer0-mistakes/blob/main/_data/roadmap.yml). Edit that file (and run `./scripts/generate-roadmap.sh` to refresh the README) to update the roadmap everywhere.
>
> _Last reviewed: 2026-04-18_

---

## Visual Timeline

The milestone timeline (a Mermaid Gantt chart) is generated at build time from `_data/roadmap.yml`, which lives in the theme repository rather than in this vault. For the current, authoritative timeline see:

- [README roadmap section](https://github.com/bamr87/zer0-mistakes#-roadmap) — the rendered timeline and milestones
- [`_data/roadmap.yml`](https://github.com/bamr87/zer0-mistakes/blob/main/_data/roadmap.yml) — the source data
- [CHANGELOG](https://github.com/bamr87/zer0-mistakes/blob/main/CHANGELOG.md) — detailed release notes

---

## Release Summary

Each milestone in `_data/roadmap.yml` carries a version, a status (✅ Completed, 🚧 In Progress, 🎯 Milestone, or 🗓 Planned), a target date, and a summary. Because that data file is maintained in the theme repository and is not part of this vault, the version table is not reproduced here.

See the full [CHANGELOG](https://github.com/bamr87/zer0-mistakes/blob/main/CHANGELOG.md) for detailed release notes.

---

## Milestone Detail

Per-version detail — status, target, summary, and feature highlights — is maintained in [`_data/roadmap.yml`](https://github.com/bamr87/zer0-mistakes/blob/main/_data/roadmap.yml) and published to the [README roadmap section](https://github.com/bamr87/zer0-mistakes#-roadmap).

## How We Prioritize

Roadmap priorities are informed by a set of signals enumerated in [`_data/roadmap.yml`](https://github.com/bamr87/zer0-mistakes/blob/main/_data/roadmap.yml).

Want to influence the roadmap? [Open a discussion](https://github.com/bamr87/zer0-mistakes/discussions) with your use case, or [propose an edit](https://github.com/bamr87/zer0-mistakes/edit/main/_data/roadmap.yml) to `_data/roadmap.yml` directly.
