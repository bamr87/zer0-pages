# Setup Wizard

The interactive setup wizard configures a new zer0-mistakes site. It runs only in development mode on the live Jekyll site — start your development server with `docker-compose up`, then open `/setup/` to walk through the configuration steps.

> [!note] Obsidian note
> The wizard is an interactive component served by the Jekyll app (`setup/wizard.html`) and does not render inside this vault. Use the running development server to access it.
