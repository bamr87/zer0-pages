# Overview

This vault is the content library of the **zer0-mistakes** Jekyll theme — a Bootstrap 5 + Jekyll starter with Docker tooling, GitHub Pages deployment, analytics, and an authoring workflow — now openable as an Obsidian vault.

## Domains

- **Getting started / Quickstart** — install the theme, set up Jekyll, GitHub, and your machine. → [[_moc/Quickstart]]
- **Docs** — the reference manual: Jekyll internals, Bootstrap, Docker, Liquid, customization, deployment, SEO, analytics, features, Obsidian integration. → [[_moc/Docs]]
- **Posts** — tutorials and articles (Jekyll, Bootstrap, Docker, accessibility, data-science experiments). → [[_moc/Posts]]
- **About** — the theme itself: authors, features, settings/config, statistics. → [[_moc/About]]
- **Notes** — cheatsheets (git, bash, docker, markdown, front matter). → [[_moc/Notes]]
- **Notebooks** — Jupyter notebooks (pandas, matplotlib, statistics, API requests). → [[_moc/Notebooks]]

Start at [[_moc/Home]].
