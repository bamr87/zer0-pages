# Overview

This vault is the content library of the **[[wiki/entities/zer0-mistakes|zer0-mistakes]]** Jekyll theme — a Bootstrap 5 + Jekyll starter with Docker tooling, GitHub Pages deployment, analytics, and an authoring workflow — now openable as an Obsidian vault. Beyond showcasing the theme, zer0-pages' declared purpose is to act as a **second brain for zer0-mistakes**: a place where research, decisions, and usage knowledge about the theme accumulate via [[wiki/entities/claude-obsidian|claude-obsidian]]'s skills and can feed back into its development.

## Domains

- **Docs** — the reference manual: Jekyll internals, Bootstrap, Docker, Liquid, customization, deployment, SEO, analytics, features, Obsidian integration (including getting-started and quickstart guides). → [[_moc/Docs]]
- **Posts** — tutorials and articles (Jekyll, Bootstrap, Docker, accessibility, data-science experiments). → [[_moc/Posts]]
- **Notes** — cheatsheets (git, bash, docker, markdown, front matter). → [[_moc/Notes]]
